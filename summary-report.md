# Shared Responsibility Model

The Shared Responsibility Model defines the security responsibilities shared between Microsoft Azure and the customer.

## Microsoft's Responsibilities

Microsoft Azure is responsible for:

- Physical security of data centers
- Hardware maintenance
- Network infrastructure
- Host operating systems
- Power, cooling, and environmental controls
- Availability of cloud services

## Customer Responsibilities

As the customer, I am responsible for:

- User account management
- Identity and access control
- Password security
- Data classification and protection
- Resource configuration
- Monitoring resource usage
- Configuring storage security settings

# Identity and Access Management

I explored Microsoft Entra ID and learned about Role-Based Access Control (RBAC), which allows administrators to assign permissions to users based on predefined roles such as:

- Owner
- Contributor
- Reader

RBAC helps organizations implement the principle of least privilege and improve cloud security.
