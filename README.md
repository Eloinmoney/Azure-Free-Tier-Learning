# Azure Free Tier Account Setup Project

## Project Overview

This project demonstrates the successful setup and exploration of a Microsoft Azure Free Tier account. The objective of this project was to gain practical experience with cloud computing fundamentals, Azure resource management, identity management, and cost monitoring.

## Project Goals

The goals of this project were to:

- Establish a functional Microsoft Azure cloud environment
- Explore the Azure Portal interface
- Understand subscriptions and resource groups
- Learn basic Identity and Access Management (IAM) concepts
- Explore Azure regions and availability zones
- Deploy a basic cloud resource
- Understand cloud cost management
- Learn about the Shared Responsibility Model

---

## Tasks Completed

### ✅ Account Registration
Created a Microsoft Azure Free Tier account and completed the verification process.

### ✅ Azure Portal Exploration
Explored the Azure Portal dashboard, search functionality, and service menus.

### ✅ Governance Setup
Created a Resource Group named:

**CloudComputing-Learning-RG**

under:

**Azure subscription 1**

### ✅ Identity Awareness
Explored Microsoft Entra ID (formerly Azure Active Directory) and reviewed Role-Based Access Control (RBAC).

### ✅ Region Selection
Selected the Azure region:

**South Africa North**

to optimize latency and service availability.

### ✅ Resource Deployment
Successfully deployed an Azure Storage Account named:

**elofirststorage**

### ✅ Cost Management
Accessed Azure Cost Analysis and explored budget and cost monitoring features.
